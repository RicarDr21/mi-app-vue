# Mi proyecto de práctica

Stack elegido: Vue (frontend)

Motivo de la elección (según los 4 criterios vistos en clase):
- Lenguaje: JavaScript/TypeScript, lenguaje que ya conozco y es ampliamente usado.
- Curva de aprendizaje: baja, sintaxis clara y poca configuración inicial.
- Facilidad de instalación: el CLI de Vue (npm create vue@latest) genera el proyecto listo para correr en segundos.
- Comunidad y documentación: Vue tiene documentación oficial completa y una comunidad activa.
